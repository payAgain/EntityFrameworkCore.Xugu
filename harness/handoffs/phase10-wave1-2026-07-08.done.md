## 交付物

### 10.001 CI/CD 实库矩阵

- `.github/workflows/ci.yml` — GitHub Actions：build + verify.ps1 -RunTests + dotnet test；matrix 覆盖 windows-latest / ubuntu-latest（linux 依赖 xugusql.so 可用性）
- `.gitlab-ci.yml` — GitLab CI：build + verify + test；secrets 经 `XUGU_TEST_CONN` 注入
- secrets 与运行说明文档化到 `docs/TESTING.md`

### 10.002 全量门禁回归

- `harness/scripts/verify.ps1` 新增 `-RunTests` 模式，串联 build + verify-source-lineage + dotnet test
- CI integration job 调用 `verify.ps1 -RunTests`，失败即阻塞合并

### 10.003 NuGet 发布流水线 2.0.0

- `harness/scripts/publish-nuget.ps1` dry-run 验证 **2.0.0**（`Version.props` 已是 2.0.0）
- GitLab `publish-nuget` manual job：手动触发，产出 `artifacts/Microsoft.EntityFrameworkCore.Xugu.2.0.0.nupkg`

### 10.004 用户文档刷新

- `docs/GETTING-STARTED.md` — 安装 / 连接串 / `UseXugu` / 迁移示例全部刷新到 2.0.0
- 新增 `docs/XUGU-VS-MYSQL.md` — XuguDB vs MySQL 方言差异用户对比（IDENTITY vs AUTO_INCREMENT、LIMIT/TOP、SYS_GUID、兼容模式等）
- `README.md` 链到 `XUGU-VS-MYSQL.md`

### 10.005 剩余测试 triage

- `harness/references/phase-10-test-triage.md` — 剩余 ~374 Pomelo 测试方法分类：
  - **P1 高价值**：Monster / Specification / Query 深覆盖 / 9.T defer → Wave 2/3

  - **P2 平台 / 性能**：Linux RID / 参数内联 → Wave 5
