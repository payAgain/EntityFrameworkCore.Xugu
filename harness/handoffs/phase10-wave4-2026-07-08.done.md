## 10.105 ROW_COUNT（blocked）

### 实库验证

启用 Pomelo 风格 `SELECT ROW_COUNT()` / `WHERE ROW_COUNT() = n` 后，SaveChanges 失败：

```
[E10049 L3 C9] 字段变量或函数"ROW_COUNT"()不存在
```

即使在 `COMPATIBLE_MODE=MYSQL` 下亦不可用。**51** 条 CRUD/Update 测试失败（全量回归）。

### 处置

- 回滚 `XuguUpdateSqlGenerator` 至 `SELECT 1` / `1 = n` 占位
- `OptimisticConcurrencyTests.Stale_concurrency_token_throws_*` 保持 skip（更新 skip 理由为 E10049）
- `docs/LIMITATIONS.md`、`sql-dialect.contract.md` 登记 blocked

### 后续解锁条件

驱动 `RecordsAffected` 可靠返回，或 XuguDB 提供等价 affected-rows API。
