## 10.205 Linux x64 RID（blocked）

### 调研结论

| 来源 | 发现 |
|------|------|
| `external/csharp-driver/` | 仅 `test_xugusql/64/xugusql.dll`（Windows）；`test_xugusql/linux-x64/` 不存在 |
| `XGCSClient.csproj` | 无 `runtimes/` 多 RID 打包 |
| `README.md` | 描述 Linux 使用需从 NuGet 安装 `XuguClient`；源码编译产出 nupkg 但仓库未含 `.so` |
| `CMakeLists.txt` | 按 OS/arch 输出路径，需完整 C++ 驱动构建链 |

### 已做预备

| 文件 | 变更 |
|------|------|
| `NativeAssets.props` | `XuguNativeSoPath` / `XuguNativeSoExists` 条件检测 |
| `src/EFCore.Xugu/EFCore.Xugu.csproj` | `runtimes/linux-x64/native/libxugusql.so` 条件打包 |

### 解锁条件

虚谷官方在 `csharp-driver` 或 `Xuguclient` NuGet 发布 `linux-x64` 原生库。
